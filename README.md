A premium Fitness Scheduler PWA (Progressive Web App) designed for iPhone.

## Features
- **Modern UI**: Dark mode with Glassmorphism aesthetic.
- **Scheduler**: Easy workout tracking with a daily view.
- **PWA**: Installable on iPhone Home Screen via Safari.
- **Performance**: Built with Vite + React for lightning-fast speeds.

## Installation for iPhone
1. Access the hosted URL in **Safari**.
2. Tap the **Share** button (bottom center).
3. Scroll down and tap **Add to Home Screen**.
4. Open FitSched from your Home Screen!

## Local Development
```bash
npm install
npm run dev
```
